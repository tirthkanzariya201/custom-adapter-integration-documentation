Use this page to confirm your app meets minimum OS requirements and dependency versions before integrating.

## Package compatibility

| TapMind package | Required GMA SDK | Min Android | Min iOS |
|-----------------|------------------|-------------|---------|
| AdMob Adapter | 25.0.0 | 23 (native), 24 (Flutter/RN) | 12.0 (native), 15.0 (Flutter/RN) |
| GAM Adapter | 25.0.0 | 23 (native), 24 (Flutter/RN) | 12.0 (native), 15.0 (Flutter/RN) |
| AppLovin Adapter | Compatible resolution (wrappers) | 23 (native), 24 (Flutter/RN) | 12.0 (native), 15.0 (Flutter/RN) |
| Unity LevelPlay Adapter | Compatible resolution (wrappers) | 23 (native), 24 (Flutter/RN) | 12.0 (native), 15.0 (Flutter/RN) |
| AdMob Adapter (GMA Next-Gen) | 25.0.0 | 23 | n/a |
| GAM Adapter (GMA Next-Gen) | 25.0.0 | 23 | n/a |
| Orchestration SDK | - | 24 | n/a |

AdMob and GAM adapters require GMA SDK **25.0.0** on Android. Confirm current minimums with your account manager before upgrading.

|-----------|-------------|---------|-------|
| Native Android | 23 | - | compileSdk 36 recommended |
