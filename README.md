# TapMind Custom Adapter SDK Integration

Welcome to the TapMind Custom Adapter integration guide. This documentation provides a seamless, step-by-step framework to integrate TapMind as a third-party ad network mediation partner.

By implementing our lightweight adapter, you can efficiently maximize ad revenue across multiple ad formats, including **Banner**, **Interstitial**, **Native**, and **Rewarded** ads.
